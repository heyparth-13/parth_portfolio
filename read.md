# Parth's Portfolio

This is the repository for my personal portfolio website. It features a modern, responsive frontend built with Next.js, React, and Tailwind CSS, and a robust backend built with Node.js and Express to handle data such as projects, skills, and contact messages. 

## Features
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB (implied by db.js)
- **Sections:** Projects, Skills, Messages/Contact
